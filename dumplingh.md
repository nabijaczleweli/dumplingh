dumplingh(1) -- Issue/PR exporter for GitHub
============================================

## SYNOPSIS

`dumplingh` [OPTIONS] <REPO_SLUG>

## DESCRIPTION

Issue/PR exporter for GitHub, which is a fancy name for dump issue and
PR data to a JSON file.

Exit values and possible errors:

    1 - I/O error
    2 - parsing error

## OPTIONS

  -v --verbose

    Print out more data.

    Default: false.
